Get Started

Include ztext.min.js on your site before the closing </body> tag on your HTML file. For example:

<script src="/path/to/ztext.min.js"></script>

Initialization

There are two ways to use ztext. Pick whichever method is easier for you.

  1. HTML attributes
  2. JavaScript

How it works

Ztext gives the illusion of volume, turning 2D HTML into 3D. There's no need to spend hours fiddling with <canvas> or forcing users to download multi-megabyte WebGL libraries.

Over 97% of users use a web browser that supports CSS 3D transforms, which ztext needs to work. In unsupported browsers, ztext gracefully turns off.

Initialize with HTML attributes

With this method, ztext will look for any HTML element with the data-z attribute and will apply a 3D effect to it.

Take this code for example:

<h1 data-z data-z-layers="3" data-z-depth=".5em">A</h1>

Structurally, ztext turns that code into this:

<h1 data-z data-z-layers="3" data-z-depth=".5em">
   <span class="z-text">
      <span class="z-layers">
         <span class="z-layer">A</span>
         <span class="z-layer">A</span>
         <span class="z-layer">A</span>
      </span>
   </span>
</h1>

The duplicate layers are visible to users but ztext makes them hidden from screen readers and other forms of user interaction as much as possible.

Initialize with JavaScript

Use ztext with vanilla JS with new Ztextify(elem, {}). This constructor accepts two arguments: the HTML element and an options object.

var element = document.querySelector('h1');
var ztxt = new Ztextify(element, {
   // Options
   depth: '30px',
   layers: 10,
   fade: true
});

Styling

Start with this CSS snippet and style ztext any way you want. It's easy to integrate with CSS animations and transitions.

.z-text {
   /* Tip: Apply CSS transforms here */
   /* rotateX() == up/down */
   /* rotateY() == left/right */
   transform: rotateY(-30deg);
   font-family: "Georgia", serif;
   transition: transform 1s;
}

[data-z]:hover .z-text {
   transform: rotateY(30deg);
}

.z-layer {
   color: #455a64;
}

.z-layer:first-child {
   color: #90a4ae;
}

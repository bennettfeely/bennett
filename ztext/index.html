Get Started

Include ztext.min.js on your site before the closing </body> tag on your HTML file. For example:

<script src="/path/to/ztext.min.js"></script>

Initialization

There are two ways to use ztext.

  1. JavaScript, or
  2. HTML attributes

Pick whichever method is easier for you.

Initalize with JavaScript

Use ztext with vanilla JS with new Ztextify(elem, {}). This constructor accepts two arguments: the HTML element and an options object.

var element = document.querySelector('h1');
var ztxt = new Ztextify(element, {
   // Options
   depth: '30px',
   layers: 10,
   fade: true
});

Initalize with HTML attributes

ztext will look for any HTML element with the data-z attribute and will apply a 3D effect to it.

Take this code for example:

<h1 data-z data-z-layers="3" data-z-depth=".5em">A</h1>

Structually, ztext turns that into this:

<h1>
   <span class="z-text">
      <span class="z-layers">
         <span class="z-layer">A</span>
         <span class="z-layer">A</span>
         <span class="z-layer">A</span>
      </span>
   </span>
</h1>

Styling

Start with the following CSS snippet and style ztext any way you want.

.z-text {
   /* Tip: Apply CSS transforms here */
   /* rotateX() == up/down */
   /* rotateY() == left/right */
   transform: rotateX(-20deg);
}

.z-layer {
   color: #455a64;
}

.z-layer:first-child {
   color: #90a4ae;
}
